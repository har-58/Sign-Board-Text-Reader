# Sign Board Text Reader

This project extracts text from signboards using OCR (EasyOCR) & OpenCV.

## Features
- Detect text from real-world signboard images
- Draw bounding boxes around recognized text
- Save extracted text into output folder

## Tech Stack
- Python
- EasyOCR
- OpenCV
- NumPy

## How to Run
```bash
pip install -r requirements.txt
python src/main.py
